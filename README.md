## Сборка проекта(рекомендую использовать yarn)

### Установка зависимостей
```sh
yarn
```

### Запуск с хот-релоадами для разработки
```sh
yarn dev
```

### Type-checking
```sh
yarn lint
```

### Сборка для production
```sh
yarn build
```
